# Rocket League Latvia Discord Bot

## Requirements

- Node.js 8+
- MongoDB

## Installation

1. Clone the project

```sh
git clone git@github.com:RocketLeagueLatvia/discord-bot.git
```

2. Install dependencies

```sh
npm install
```

3. Copy `config.json.dist` to `config.json` and edit the contents.

4. Run `index.js` script.
